Задача
Расширить данные базового источника (регистр «Проект бюджета») информацией из таблицы сопоставления для формирования полного набора данных,
необходимого для дальнейшего построения отчета.

Решение
Разработан новый источник данных, объединяющий сведения из регистра «Проект бюджета» и таблицы сопоставления. 
Решение предусматривает гибкую настройку:
	полей, используемых для сопоставления данных;
	состава атрибутов, которыми дополняется базовый источник данных.
