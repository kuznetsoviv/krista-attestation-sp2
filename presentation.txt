Расширенный источник данных
	Задача
		Расширить данные базового источника (регистр «Проект бюджета») информацией из таблицы сопоставления для формирования полного набора данных,
		необходимого для дальнейшего построения отчета.
	Решение
		Разработан новый источник данных, объединяющий сведения из регистра «Проект бюджета» и таблицы сопоставления. 
		Решение предусматривает гибкую настройку:
			полей, используемых для сопоставления данных;
			состава атрибутов, которыми дополняется базовый источник данных.

Рефакторинг операций блока РРО

Проблемы текущей реализации:
	использование фреймворка Drools
	использование системы контроля версий Mercurial
	наличие значительного объёма legacy-кода
	предметно-ориентированная логика вынесена в отдельные репозитории, что усложняет сопровождение и развитие системы

Решение:
	переход на Kotlin
	миграция на Git
	использование актуальной и поддерживаемой кодовой базы
	выделение и централизация конфигурируемых настроек для повышения гибкости и упрощения сопровождения
